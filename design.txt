API:

POST lesson

    - Input:
        - Copy-pasted textbook chapter
        - Categories of explanation
            - Name
            - Short description
            - Media related to the category

GET lesson/{id}

POST lesson/{id}/student-prediction

    - Input: one or more of
        - Text
        - Image (handwriting or drawing)

    - Logic:
        - If image, ask Google Vision API to describe the image in text
        - Ask the LLM which category it belongs in and why
        - Save the prediction (the student input + the LLM response)
